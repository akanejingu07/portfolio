# Portfolio Site

エメラルドグリーンを基調とした、  
**静謐で神聖な雰囲気のポートフォリオサイト**です。

作品そのものが主役になるよう、  
余白・タイポグラフィ・控えめなアニメーションを重視して設計しています。

---

## URL

https://akanejingu07.github.io/portfolio/

---

## コンセプト

- Sacred / Quiet / Emerald  
- 主張しすぎないが、記憶に残る
- 触れるとわずかに反応する静かな演出
- 情報量より「空気感」を大切にする

---

## ページ構成

- **Top**  
  メインビジュアルと世界観の提示  
  エメラルドグリーンのきらめきエフェクトを全体に配置

- **About**  
  自己紹介、仕事スタンス、価値観、使用技術などを整理

- **Works / Gallery**  
  余白を活かした作品一覧  
  画像は最適化し、クリックで全体表示できる構成

---

## 実装している演出

- エメラルドグリーンの微細なきらめき（JavaScript）
- 全ページ共通の控えめなアニメーション
- マウス操作に反応する一瞬の光
- スクロールや操作の邪魔をしない設計

---

## 使用技術

- HTML5  
- CSS3  
- JavaScript（Vanilla JS）
- Google Fonts（Noto Serif JP）
- GitHub Pages（静的サイト公開）

※ フレームワークは使用せず、基礎技術のみで構成しています。
